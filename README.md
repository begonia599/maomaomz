⚠️ **本插件免费提供，禁止倒卖** ⚠️

## 🔒 安全说明

本插件代码完全开源，安全透明，用户可自行审计。

### ✅ 我们的安全承诺

| 安全特性 | 说明 |
|---------|------|
| **无跨窗口通信** | 不使用 `postMessage`，不监听全局 `message` 事件 |
| **无外部 API 暴露** | 没有任何可被外部网页/iframe 调用的接口 |
| **本地数据存储** | 用户的 API Key 和设置仅存储在浏览器 localStorage |
| **输入验证** | 对授权码、API 端点等用户输入进行格式验证，防止注入攻击 |
| **HTTPS 通信** | 所有外部请求（授权验证、版本检查）均使用 HTTPS 加密 |

### 🛡️ 与某些插件的区别

部分插件可能存在以下安全风险（本插件**均不存在**）：

- ❌ 监听 `window.addEventListener('message')` 但不验证来源
- ❌ 使用 `postMessage(..., '*')` 广播敏感数据
- ❌ 暴露 `triggerSlash`、`getChatMessages` 等敏感 API 给外部调用

**本插件不存在上述任何风险。**

### 📡 网络请求说明

本插件仅在以下情况发起网络请求：

1. **授权验证** - 发送到 Cloudflare Worker（HTTPS）
2. **版本检查** - 从 GitHub API 获取最新版本号
3. **AI 总结** - 发送到用户自己配置的 API 端点

所有请求均不会泄露用户的聊天内容到第三方服务器。
